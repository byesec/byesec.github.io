拖延下的仓促开始

2021年12月31日

开始

现在是2021年12月31日22点31分,从几天前的butterfly主题配置到今天还是决定使用Next,原因有二,一是在配置butterfly中图片的清晰程度和加载速度二者矛盾让我难以平衡,二是看了许多hexo博客后,发现还是Next的简洁更舒服些。
耗时一天,参考了不少文章后,下一步备案绑定域名吧,这大概是一个能拿得出来的样式了吧哈哈。
上面提到了时间,再过一个小时就要2022年了,将至的这一年还是不立flag了,也许开始写博客就意味着新的开始吧,下面的配置还没时间去整理格式了，这样挺有意思哈哈.

安装git

安装node.js

安装hexo

修改hexo的_config.yml

修改themes/next/_config.yml

npm install hexo-generator-search –save

npm install hexo-generator-search –save
npm install hexo-generator-searchdb –save

yum install -y nginx
chmod 775 -R /home/blog/
vim /etc/nginx/nginx.conf
server_name xxxx; //这里换成你自己的域名

root /home/blog/bysec/public/; //这里换成你刚才创建的目录
systemctl start nginx.service

systemctl restart nginx.service

cp themes/next/_config.yml _config.next.yml

在 themes/next/_vendors.yml 中添加 darkmode的 cdn：

darkmode_js:
name: darkmode-js
version: 1.5.7
file: lib/darkmode-js.min.js
